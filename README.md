# chess_game
blender game

A 3D chess simulator built with Blender and powered by UPBGE (Blender Game Engine).

«This project is a personal summer project focused on learning game development, Python scripting, and real-time logic in UPBGE.»

✨ Features

- ♟️ Fully modeled 3D chess board
- 🎮 Real-time gameplay using UPBGE
- 📷 Smooth camera controls
- 🐍 Python scripting for game logic
- 🌧️ Realistic environment and visual effects
- 💡 Expandable architecture for AI and multiplayer

---

🛠️ Built With

- Blender
- UPBGE
- Python
- GLSL Shaders

🚀 Project Goals
